# Washington_House_Sales_Dashboard# 🏠 King County House Sales Dashboard

## 📌 Project Overview
This project presents an interactive Tableau dashboard that analyzes house sales data from King County, Washington. The dashboard provides insights into house prices, property characteristics, geographical distribution, and market trends to support data-driven decision-making.

---

## 📊 Dashboard Features

### 🔹 Sales Price by Date
- Visualizes average house sale prices over time.
- Identifies trends and fluctuations in the housing market.

### 🔹 Average Sales Price by Location
- Interactive map showing average property prices across different regions.
- Helps identify high-value and low-value areas.

### 🔹 House Price Distribution
- Histogram displaying the distribution of house prices.
- Highlights pricing patterns and outliers.

### 🔹 Bedrooms Distribution
- Analyzes the frequency of properties based on the number of bedrooms.

### 🔹 Bathrooms Distribution
- Shows the distribution of houses by bathroom count.

### 🔹 View vs Condition Heatmap
- Compares house prices based on property condition and view quality.
- Identifies factors influencing property values.

### 🔹 Interactive Filters
Users can filter data by:
- Month and Year
- Year Built
- Average Square Foot Living Area
- Average Square Foot Lot Size

## 📈 Key Insights

- Most houses are priced between $300K and $600K.
- Properties with better views and conditions generally have higher sale prices.
- Houses with 3–4 bedrooms are the most common.
- Certain regions in King County consistently show higher average sale prices.
